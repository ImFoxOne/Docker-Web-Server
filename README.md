# Dockerized Nginx Web Server 🐳

This is my first containerization project. I used Docker to containerize a simple static website hosted on an Nginx web server.

## Features
* **Containerization:** Built and packaged a web server using Docker.
* **Custom Web Page:** Replaced the default Nginx page with a custom HTML file.
* **Port Mapping:** Mapped host ports to the container to serve the website locally.

## Tech Stack
* Docker
* Nginx
* Linux (Ubuntu)
